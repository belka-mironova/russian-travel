# Project:  "Russian-Travel"


## Created as one of the [Yandex.Praktikum](https://praktikum.yandex.com/) Bootcamp projects

Russian-Travel is a single page website about travelling across Russia.


Adaptive design
Flexbox
Grid Layout
Positioning Elements

1. Clone the repo
